勝率カウンター

対戦ゲームの勝ち負けをすばやく記録し、チーム・デッキ・キャラごとの勝率を確認できるiOSアプリです。

勝 負 62.5%

基本的な使い方

  • カウンターを作成し、名前と任意の写真を登録します。
  • 試合後に「勝」または「負」を押すだけで記録できます。
  • 勝率、勝ち数、負け数、合計試合数は自動で更新されます。
  • ホーム画面・ロック画面ウィジェットからも勝敗を記録できます。

よくある確認事項

  • ウィジェットに表示するカウンターは、アプリ内の設定画面から割り当てます。
  • 誤って記録した場合は、アプリ内の履歴から削除できます。
  • 勝/負 と W/L の表記は、設定画面から切り替えできます。
  • データは端末内に保存されます。ログインは不要です。
