プライバシーポリシー

最終更新日: 2026年7月23日

勝率カウンターは、対戦ゲームの勝敗記録と勝率管理を目的としたiOSアプリです。 本アプリは、ログインを必要とせず、記録データを端末内に保存します。

収集する情報

本アプリは、個人を特定する情報を収集しません。 カウンター名、勝敗履歴、登録写真などのデータは、ユーザーの端末内に保存されます。

写真へのアクセス

ユーザーがカウンターに写真を設定する場合のみ、写真ライブラリまたはカメラへのアクセス許可を求めます。 選択または撮影された写真は、カウンター表示のために端末内で使用されます。

第三者への提供

本アプリは、ユーザーの記録データや写真を外部サーバーへ送信せず、第三者へ提供しません。

お問い合わせ

本ポリシーやアプリに関するお問い合わせは、 GitHub Issues からご連絡ください。

変更について

本ポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、このページ上でお知らせします。
