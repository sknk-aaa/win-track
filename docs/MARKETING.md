# MARKETING — 勝率カウンター

## 基本訴求

勝ったら勝ち、負けたら負け。アプリを開かずウィジェットから勝率を記録できる、対戦ゲーム向けの勝率カウンター。

## 想定ユーザー

- 対戦ゲームでチーム、デッキ、キャラごとの勝率を見たい人。
- 試合後にメモアプリや表計算へ記録するのが面倒な人。
- ホーム画面やロック画面からすぐ記録したい人。

## ストア訴求案

- ウィジェットから1タップで勝敗記録。
- チーム別・デッキ別・キャラ別に勝率管理。
- 履歴からあとで見返し、不要な記録は削除可能。
- ログイン不要、端末内保存。

## App Store提出メタデータ案

Apple公式の要件に合わせ、App Store名/サブタイトルは30文字以内、キーワードは100文字以内、プロモーションテキストは170文字以内に収める。
検索語は、App Store名とサブタイトルに強い語を置き、キーワード欄では重複を避ける。

### 推奨セット

App Store名:
```text
勝率カウンター - 対戦ゲーム
```

ホーム画面表示名:
```text
勝率カウンター
```

サブタイトル:
```text
ウィジェットで勝ち負けを即記録
```

キーワード:
```text
戦績,履歴,集計,デッキ,チーム,キャラ,バトル,レート,FPS,カード,格闘,MOBA,eスポーツ,スコア,対人,勝敗,試合,マッチ,ランク,勝数,負数,連勝,管理,ボード,麻雀,将棋,TCG
```

プロモーションテキスト:
```text
対戦ゲームの勝ち負けを、アプリやウィジェットからすぐ記録。チーム、デッキ、キャラごとの勝率をシンプルに管理できます。
```

説明:
```text
勝率カウンターは、対戦ゲームの勝ち負けをすばやく記録し、チーム・デッキ・キャラごとの勝率を確認できるシンプルな記録アプリです。
ホーム画面やロック画面のウィジェットからも記録できるので、試合後にアプリを探して開く手間を減らせます。

こんな人におすすめ
・デッキやチームごとの勝率を見たい
・勝ち/負けだけを素早く残したい
・試合後にメモアプリや表計算へ入力するのが面倒
・ホーム画面やロック画面からすぐ記録したい
・複数のキャラ、編成、ルールごとに成績を分けて管理したい

主な機能
・複数のカウンターを作成
・勝ち/負けを1タップで記録
・勝率、勝ち数、負け数、合計試合数を自動計算
・ホーム画面ウィジェットから記録
・ロック画面ウィジェットから記録
・履歴一覧とカウンター別の絞り込み
・誤って記録した履歴の削除
・名前と写真でカウンターを管理
・勝/負 と W/L の表記切り替え

使い方はシンプルです。
カウンターを作成して、試合が終わったら「勝」または「負」を押すだけ。勝率や試合数は自動で更新されます。
よく使うカウンターはウィジェットに割り当てておけば、アプリを開かずに記録できます。

データはこの端末内に保存されます。ログインは不要です。
```

このバージョンの狙い:
- `勝率カウンター` と `対戦ゲーム` をApp Store名に置き、検索意図を直撃させる。
- サブタイトルは、最大の差別化である `ウィジェット` と行動語の `即記録` を入れる。
- キーワード欄は、タイトル/サブタイトルに入っていない利用シーン、ゲームジャンル、管理語を詰める。
- 説明文の冒頭2行で「誰向け」「何が便利」「なぜ入れるか」を伝える。

### 代替案

よりシンプル:
```text
App Store名: 勝率カウンター
サブタイトル: 対戦ゲームをウィジェットで記録
```

より戦績管理寄り:
```text
App Store名: 勝率カウンター - 戦績管理
サブタイトル: 対戦ゲームの勝ち負けを記録
```

提出時は、推奨セットを第一候補にする。

## スクリーンショット方針

1. カウンター一覧で複数チームの勝率が見える画面。
2. ホーム画面ウィジェットから勝ち/負けを記録する画面。可能なら複数枠を見せる。
3. ロック画面ウィジェットから記録する画面。勝率、勝敗数、円形の勝ち/負けボタンが分かる状態にする。
4. 履歴ページで全記録を確認する画面。
5. 設定画面でウィジェット枠、勝敗表記、アプリアイコンを調整できる画面。
