# MARKETING — 勝率カウンター

## 基本訴求

勝ったら勝ち、負けたら負け。アプリを開かずウィジェットから勝率を記録できる、対戦ゲーム向けの勝率カウンター。

## 想定ユーザー

- 対戦ゲームでチーム、デッキ、キャラごとの勝率を見たい人。
- 試合後にメモアプリや表計算へ記録するのが面倒な人。
- ホーム画面やロック画面からすぐ記録したい人。

## ストア訴求案

- ウィジェットから1タップで勝敗記録。
- チーム別・デッキ別・キャラ別に勝率管理。
- 履歴からあとで見返し、不要な記録は削除可能。
- ログイン不要、端末内保存。

## スクリーンショット方針

1. カウンター一覧で複数チームの勝率が見える画面。
2. ホーム画面ウィジェットから勝ち/負けを記録する画面。
3. ロック画面ウィジェットから記録する画面。
4. 履歴ページで全記録を確認する画面。
5. 写真付きカウンター作成画面。
