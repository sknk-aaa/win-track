# AGENTS.md — 勝率カウンター

## 概要

勝率カウンターは、対戦ゲームなどの勝ち/負けをアプリ本体とiOSウィジェットから記録し、勝率・勝敗数・履歴を確認するiOSアプリ。

## 固有ルール

- iOSのみ。Android対応はしない。
- ビルド/配信は親プロジェクトの共通docに従う。このリポ内に重複して書かない。
- 仕様の正は `docs/DESIGN.md`。
- ウィジェットからの記録後、表示を即時反映する体験を最重要仕様として扱う。
- ウィジェットには写真を表示しない。
- 取り消し・個別削除はアプリ内のみ。

## Docs

- `docs/DESIGN.md` — 仕様、画面、データモデル
- `docs/OPERATIONS.md` — bundle id、Secrets、ビルド/配信参照
- `docs/HANDOFF.md` — 現状、次タスク、既知の問題
- `docs/MARKETING.md` — ストア訴求、スクショ方針、後回し施策
